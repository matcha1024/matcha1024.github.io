解き方

毎回 'fox' を消して、はじめからもう一度見る…を繰り返していては、間に合いません。

個の都市がある。
都市 から都市 まで移動するには、 の時間がかかる。
都市から出発し、すべての都市を一度ずつ訪れ、また都市に戻ってくるとき、

制約



解き方

移動は、 → x → x ... →

ここで求めるものは、 と の間の通り数です。
例えば だと、間は の順列になります。

ということで順列全探索をします。
C++では、

do{
    処理
}while(next_permutation(v, v+n));

という形で順列全探索ができます。
順列全探索は なので が を超えると厳しいのですが、

#define forout(out) for(int i=0; i<(int)out.size(); i++) cout<<out[i]<<endl

AtCoder AGC 033 A - Darker and Darker

AtCoder ３００点

問題について

AGC 033 A - Darker and Darker
３００点
difficulty:1245

問題概要

行 列からなるマス目において、
すべての黒マス('#')の上下左右の白マス('.')を黒マスにする。
すべてのマスを黒マスにするには、この操作を何回すればよいか。

制約

解き方

スタート地点が複数あるBFSをします。
そうすることによって、それぞれの白マスの
一番近い黒マスからの距離を求めることができます
まず、初期状態での黒マスの座標を に入れてから、
普通にBFSをすればよいです。

#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
typedef pair<int, int> P;
#define forin(in) for(int i=0; i<(int)in.size(); i++) cin>>in[i]
#define forout(out) for(int i=0; i<(int)out.size(); i++) cout<<out[i]<<endl
#define rep(i, n) for(int i=0; i<(n); i++)
template<class T> inline bool chmax(T& a, T b) { if (a < b) { a = b; return 1; } return 0; }
template<class T> inline bool chmin(T& a, T b) { if (a > b) { a = b; return 1; } return 0; }
 
int main(){
    const int dx[]={1, -1, 0, 0};
    const int dy[]={0, 0, 1, -1};
 
    int H, W; cin>> H >> W;
    vector<string> A(H);
    forin(A);
 
    vector<vector<int>> dist(H, vector<int>(W, -1));
    queue<P> que;
 
    rep(i, H)rep(j, W){
        if(A[i][j] == '#'){
            dist[i][j] = 0;
            que.push(P(i, j));
        }
    }
 
    while(!que.empty()){
        P p = que.front();
        que.pop();
 
        rep(i, 4){
            int next_y = p.second + dy[i];
            int next_x = p.first + dx[i];
 
            if(next_x<0 || next_x>=H || next_y<0 || next_y>=W)continue;
 
            if(dist[next_x][next_y] != -1)continue;
 
            dist[next_x][next_y] = dist[p.first][p.second] + 1;
            que.push(P(next_x, next_y));
        }
    }
 
    int ans = 0;
    rep(i, H)rep(j, W){
        chmax(ans, dist[i][j]);
    }
    cout<< ans << endl;
}

AtCoder ARC 104 B - DNA Sequence

AtCoder ４００点

問題について

ARC 104 B - DNA Sequence
400点 difficulty:410

問題概要

'A' 'T' 'C' 'G' からなる文字列 の部分文字列 を並び替えた文字列
において、すべての と の組み合わせが
('A', 'T') または ('C', 'G') にできるものの個数を求める。

制約

解き方

つまり、 に含まれる 'A' の数と 'T' の数、
'C' の数と 'G' の数がそれぞれ等しければよいわけです。

例えば、 'ACGT' は、A:1 | T:1 | C:1 | G:1 なので条件を満たします。
しかし、 'ACGA' は A:2 | T:0 | C:1 | G:1 なので条件を満たしません。

愚直にループを回すと、 なのでTLEになります。

そこで、累積和を使います。
'A' の数の累積和、 'T' の数の累積和、 'C' の数の累積和、 'G' の数の累積和
を用意します。
そして、2重ループで部分文字列をすべて見ていけばよいです。

#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
typedef pair<int, int> P;
#define forin(in) for(int i=0; i<(int)in.size(); i++) cin>>in[i]
#define forout(out) for(int i=0; i<(int)out.size(); i++) cout<<out[i]<<endl
#define rep(i, n) for(int i=0; i<(n); i++)
 
int main(){
    int N;
    cin>> N;
    string S;
    cin>> S;
 
    vector<int> sum_A(N+1, 0), sum_T(N+1, 0), sum_C(N+1, 0), sum_G(N+1, 0);
    for(int i = 0; i < N; i++){
        int cnt_A=0, cnt_T=0, cnt_C=0, cnt_G=0;
        if(S[i] == 'A') cnt_A++;
        else if(S[i] == 'T') cnt_T++;
        else if(S[i] =='C') cnt_C++;
        else if(S[i] =='G') cnt_G++;
        sum_A[i + 1] = sum_A[i] + cnt_A;
        sum_T[i + 1] = sum_T[i] + cnt_T;
        sum_C[i + 1] = sum_C[i] + cnt_C;
        sum_G[i + 1] = sum_G[i] + cnt_G;
    }
 
    int cnt = 0;
    for(int i = 2; i <= N; i++){
        for(int j = 0; j < N - i + 1; j++){
            if(sum_A[j + i] - sum_A[j] == sum_T[j + i] - sum_T[j] && sum_C[j + i] - sum_C[j] == sum_G[j + i] - sum_G[j]) cnt++;
        }
    }
    cout<< cnt << endl;
}

AtCoder ABC 032 B - 高橋くんとパスワード

AtCoder ２００点

問題について

ABC 032 B - 高橋くんとパスワード
200点
difficulty:454

問題概要

文字列 の、長さ の部分文字列は
何種類ある?

制約


解き方

こういう問題では、重複を許さない配列、set を使います。
部分文字列をすべてsetにinsertしていき、最後にsizeを出力すればよいです。

部分文字列は、substrを使います。

string str = "ABCDEF";
cout<< str.substr(2, 3) << endl;

>> CDE

というように、

str.substr(最初, 文字数)

と、使います。

B問題から茶diffが出てた昔のコンテストはすごいですなぁ
ただ、setを知っていれば茶diffの難しさは無いと思います。

#include<bits/stdc++.h>
using namespace std;
typedef long long ll;
typedef pair<int, int> P;
#define forin(in) for(int i=0; i<(int)in.size(); i++) cin>>in[i]
#define forout(out) for(int i=0; i<(int)out.size(); i++) cout<<out[i]<<endl
#define rep(i, n) for(int i=0; i<(n); i++)
 
int main(){
    string s;
    cin>> s;
    int k;
    cin>> k;
 
    set<string> res;
 
    if(s.size() >= k) {
        rep(i, s.size() - k + 1) {
            res.insert(s.substr(i, k));
        }
    }
    cout<< res.size() << endl;
}
